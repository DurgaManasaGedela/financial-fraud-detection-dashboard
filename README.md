# Financial Fraud Detection Dashboard

This project presents a Power BI dashboard built to analyze financial transactions and identify fraud patterns.

## Project Overview
The dashboard provides insights into transaction behavior and fraudulent activities using interactive visualizations.

## Key Metrics
- Total Transactions
- Total Transaction Amount
- Fraud Transactions
- Fraud Rate

## Insights Provided
- Fraud trends over time
- Top fraudulent transactions
- Fraud distribution by transaction type
- Transaction amount analysis

## Tools Used
- Power BI
- Data Visualization
- Data Analysis
